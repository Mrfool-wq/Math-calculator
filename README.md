# Simple Math — Beginner to Intermediate

This small site teaches and calculates basic-to-intermediate math with short step-by-step explanations.

- index.html — main page and UI
- styles.css — simple styling
- script.js — calculators and explanation steps

# create files (index.html, styles.css, script.js, README.md) with the content above

git branch -M main
git remote add origin https://github.com/Mrfool-wq/Math-Calculator.git
git push -u origin main
